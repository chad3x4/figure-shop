## Requirement:
#### Python 3.11.2
#### Django 4.2.2
#### Django CKEditor 6.6.1
#### REST Framework 3.14.0
#### Pillow 9.5.0


## How to run server:

#### 1. Clone repo
#### 2. Open in Pycharm
#### 3. Create a new schema named "fshopdb" with:
###### - Charset: "utf8-mb4"
###### - Collation: "utf8mb4_unicode_ci"
#### 4. In terminal:
###### - 'cd figure_shop'
###### - 'python manage.py makemigrations figures'
###### - 'python manage.py migrate'
###### - 'python manage.py runserver'
